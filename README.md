# ⚽ Sport Management System

A full-stack web application for managing sports competitions, teams, players, matches, and rankings. The platform provides an intuitive admin dashboard to simplify tournament and league management.

## ✨ Features

* 📊 Dashboard with key statistics
* ⚽ Sports management
* 👥 Player management
* 🛡️ Team management
* 🗓️ Match scheduling
* 🏆 Match results
* 📈 Rankings & standings
* 🔐 Admin panel for system management

## 🛠️ Tech Stack

* **Backend:** Laravel
* **Frontend:** React + Vite
* **Database:** MySQL
* **Styling:** Tailwind CSS
* **Version Control:** Git & GitHub

## 🚀 Installation

```bash
git clone https://github.com/ridaelbalaouy2004/Sport-Management.git

cd Sport-Management

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve

npm run dev
```
